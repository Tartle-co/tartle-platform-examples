This is an example of a
simple TARTLE endlink flow using widgets.

This will take you through a couple of pages simulating a flow that collects some information, although this flow can do whatever you need it to do. The important thing is that the landing page (in this case this page) has the following script in the head of the document:

<script src="https://source.tartle.co/scripts/endlink-landing.js"></script>

Inspect the source of this page to see it. And if you check your localStorage, you'll see an entry for tartle-endlink-token if you landed here from a TARTLE packet with this endlink.
